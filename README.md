# Solutions for AI SE Image Gallery/Carousel Tasks

This repo stores the solutions for the image gallery/carousel tasks in Sprint 3. The initial starting code for the series is found here:

  https://github.com/tripleten-com/ai-se_image-gallery

## Lesson Map

The lessons are spread out among several chapters.

- Chapter 5 (Responsive Design) includes lesson 1
- Chapter 6 (Grid) includes lessons 2 and 3
- Chapter 7 (Positioning) includes lessons 4 and 5

The remaining two lessons are optional and are referred to in the optional confirmation modal task for Project 1 Part 5 at the end of Sprint 3.
